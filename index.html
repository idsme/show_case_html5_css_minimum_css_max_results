Inline Text Elements

Use <code> for inline code like var x = 42;

Use Ctrl + S to save — the <kbd> tag styles keyboard shortcuts.

Use <mark> to highlight important text in a paragraph.

Use <abbr> for abbreviations like HTML and CSS.

Use <ins> and <del> to show removed content and added content in revisions.

Use <sup> for superscript like E=mc2 and <sub> for subscript like H2O.
