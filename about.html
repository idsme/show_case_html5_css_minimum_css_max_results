About

What is this project?

This is a minimal CSS stylesheet that transforms any plain HTML5 page into something presentable. It targets standard HTML tags directly — no classes needed for most elements.

The design token system in main.css uses CSS custom properties with BEM naming (--block__element--modifier) so you can re-theme the entire site by changing a handful of variables.

Features

  • Zero dependencies — no CDN, no build tools
  • Light and dark mode with toggle
  • Styles for all common HTML5 elements
  • Responsive layout
  • Accessible markup with ARIA attributes
  • BEM-named design tokens for easy theming
