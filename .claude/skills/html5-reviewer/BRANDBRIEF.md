# Brand Brief — Ids Achterhof / ACMELabs

## Brand Identity

**Person:** Ids Achterhof
**Brand:** ACMELabs
**Tagline:** "Lets Create Something Out of Nothing"
**Role:** Independent solutions architect with 30+ years building enterprise systems
**Location:** The Netherlands

## Logo

**Primary mark:** Erlenmeyer flask with cloud — represents experimentation and creation from nothing.

**Wordmark:** "ACME LaBs" — mixed-case with lowercase 'a' in "LaBs".
- Primary font: **Acme Bold** (by Mojomox) — geometric sans-serif, wide letter-spacing
- Secondary font: **Montserrat Regular** — tagline text

**Logo variants available:**
- `Acmelabs_Logo_word_org.png` — standard wordmark
- `Gemini_Improved_ACME_logo_XXL.png` — enlarged version
- `ACMELabs_logo_bottle.svg` / `.png` — flask icon only
